# Privacy Policy — Max FDP Calculator

**Last updated:** March 2026

## Overview
Max FDP Calculator is a reference tool for calculating maximum flight duty periods. 

## Data Collection
This app does not collect, store, transmit, or share any personal data or usage information. All calculations are performed locally on your device.

## Third-Party Services
This app does not use any third-party analytics, advertising, or tracking services.

## Data Storage
No user data is stored on any external server. All inputs are held in memory during use and are not retained when the app is closed.
